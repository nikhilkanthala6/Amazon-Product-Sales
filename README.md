# Amazon Product Sales Analysis

## Project Overview

This project focuses on performing Exploratory Data Analysis (EDA) on Amazon product sales data using Python. The goal of this project is to clean, preprocess, analyze, and visualize the dataset to discover meaningful insights about products, reviews, pricing, ratings, discounts, and customer behavior.

This project helped improve my practical skills in:

* Data Cleaning
* Data Preprocessing
* Exploratory Data Analysis (EDA)
* Feature Engineering
* Data Visualization
* Working with real-world datasets
